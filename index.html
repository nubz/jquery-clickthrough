What is jquery.clickthrough.js?

  • A tiny jQuery plugin that:
  • converts a collection of HTML slides into a presentation
  • that can be 'clicked' through
  • to either reveal list items one by one and/or
  • whole slides one by one

What can it contain?

  • Slides, which are HTML elements with a common CSS class
  • they can contain any HTML content including images and video
  • for example:
  • $('div.slide') is a jQuery collection of all divs to be used as slides
  • it's as simple as $('div.slide').clickthrough();
